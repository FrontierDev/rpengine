Slash Commands
======

This guide lists the available slash commands for the RPEngine addon. To use a command, type `/rpe <command>` in the chat window. Some commands accept additional arguments, which are indicated in angle brackets.

Core Commands
---------
- `sheet` - Opens the RPE Character Sheet UI.
- `ruleset` - Opens the Ruleset editor UI.
- `dataset` - Opens the Dataset management UI.
- `event` - Opens the Event window UI.
- `sginvite <PLAYER>` - Invites the player (and their party) to your current supergroup.

Secondary Feature Commands
---------
- `chanter` - Opens the Chanter configuration UI.

Debug Commands
---------
- `stat <STAT_ID>` - Prints the current value of the specified STAT_ID to the chat window.
- `leader` - Tells you whether you are the supergroup leader or not.
- `clipboard` - Opens the RPE Clipboard UI for copying/pasting data.
