Settings
========

Configuration options for the DM: encounter rules, XP scaling, loot tables, and ambient difficulty.

- Encounter rules
- XP and reward scaling
- Loot tables and distribution rules
Event Settings
==============

Global settings for events, such as turn timers, UI behavior, and default palettes.

- Turn timing and auto-advance
- UI visibility rules
- Default palette and theme

These settings are exposed in the Event Control UI and persisted in the profile database.
