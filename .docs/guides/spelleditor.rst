Spell Editor
=============

The Spell Editor allows you to create and configure spells and abilities in your dataset.

Overview
--------

Spells define magical abilities, special powers, and class features available to characters.

Accessing the Editor
--------------------

1. Open the Dataset Editor
2. Go to the **Spells** tab
3. Search or browse existing spells
4. Click **New Spell** to create

Creating a Spell
----------------

1. Click **New Spell** button
2. Fill in basic information:

   - **Name**: Spell name
   - **Level**: Spell level/tier
   - **School**: Magic school (Evocation, Abjuration, etc.)
   - **Description**: What the spell does

3. Configure casting information:

   - **Casting Time**: Action, bonus action, reaction, etc.
   - **Range**: Touch, Self, distance in feet
   - **Duration**: Instantaneous, concentration, permanent
   - **Components**: Verbal, Somatic, Material

4. Set requirements:

   - **Class**: Which classes can cast
   - **Proficiency**: Required proficiency
   - **Level Requirement**: Minimum character level

5. Configure effects:

   - **Damage/Healing**: Amount and type
   - **Saving Throw**: DC and type if applicable
   - **Special Effects**: Custom effects

6. Set cooldown:

   - **Cooldown Duration**: How long before reuse
   - **Charges**: Uses per rest, if applicable

7. Save the spell

Spell Categories
----------------

**Cantrips**
- At-will spells
- No material cost
- Level 0

**Leveled Spells**
- Require spell slots
- Limited uses per rest
- Scale with spell level

**Abilities**
- Class-specific powers
- Special features
- Custom mechanics

**Rituals**
- Cast without using spell slots
- Longer casting time
- Often utility-focused

Tips
----

- Balance spell power with casting cost
- Document all mechanical effects
- Test spells in combat scenarios
- Consider interactions with other abilities
