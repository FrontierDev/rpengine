Stats Editor
=============

The Stats Editor allows you to create and configure the ability scores and statistics for your game system.

Overview
--------

Statistics form the foundation of character mechanics. Each game system has its own set of core stats.

Accessing the Editor
--------------------

1. Open the Dataset Editor
2. Go to the **Stats** tab
3. Select or create stats for your system

Creating a Stat
---------------

1. Click **New Stat**
2. Enter the stat name (e.g., "Strength", "Intelligence")
3. Configure the following:

   - **Abbreviation**: Short code (e.g., "STR")
   - **Description**: What this stat represents
   - **Min Value**: Minimum possible value
   - **Max Value**: Maximum possible value
   - **Base Value**: Default starting value

4. Save the stat

Stat Types
----------

**Ability Scores**
- Core attributes like Strength, Dexterity, Constitution, etc.
- Typically range from 1-20 or 3-18
- Used as basis for skill checks and saves

**Derived Stats**
- Calculated from ability scores
- Examples: Armor Class, Hit Points, Initiative
- Update automatically when base stats change

**Resources**
- Used for tracking limited uses
- Examples: Spell Slots, Hit Dice, Action Points
- Refresh at specific intervals

Best Practices
--------------

- Use consistent naming conventions
- Clearly define the purpose of each stat
- Set appropriate min/max values for your system
- Add helpful descriptions for custom stats
